WorkforceIS

To run the application, one must login to SQL*Plus and use the provided scripts createDatabase.sql to create the necessary tables and insertTestData.sql to populate the database with its initial data. Once that is complete, the applications can be run from the dbapp packages, which contain the packages dbaccess and dbgui. Once all packages are compiled, the program can be started from the WorkforceIS class.
